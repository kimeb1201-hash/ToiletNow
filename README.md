# 🚽 Toilet Now

Toilet Now는 사용자의 현재 위치를 기반으로 주변 화장실을 빠르게 찾을 수 있도록 도와주는 위치 기반 서비스입니다.

## 📌 프로젝트 소개
급하게 화장실이 필요한 상황에서 주변 화장실의 위치와 기본 정보를 확인할 수 있도록 기획한 웹 서비스입니다.

## 🎯 주요 기능
- 현재 위치 기반 화장실 탐색
- 사용자 참여 기반 정보 제공 (혼잡도, 청결도 등)
- EarnLearning OAuth 로그인 연동 시도

## 🚧 현재 진행 상태
